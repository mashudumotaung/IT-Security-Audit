# IT Security Audit Checklist

## System Security Review

### 1. Password Strength
I reviewed the passwords used on my accounts to ensure they are strong. 
Strong passwords include uppercase letters, lowercase letters, numbers, and special characters.

Recommendation:
Use unique passwords and enable two-factor authentication where possible.

### 2. Firewall Status
The Windows Firewall was checked through Windows Security settings.

Result:
Firewall is enabled and protecting the system from unauthorized access.

### 3. Software Updates
The system was checked for updates using Windows Update.

Result:
The latest updates and security patches were installed.

### Conclusion
Performing a basic security audit helps ensure that the computer is protected from cyber threats through strong passwords, firewall protection, and updated software.
